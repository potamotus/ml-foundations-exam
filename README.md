# ML Foundations — Exam Trainer (ОМО, ВШЭ)

Интерактивный тренажёр для подготовки к **теоретическому** экзамену по курсу
«Основы машинного обучения» (НИУ ВШЭ). Один самодостаточный HTML-файл, работает
офлайн, прогресс сохраняется в браузере (`localStorage`).

**▶ Живая версия:** https://potamotus.github.io/ml-foundations-exam/

## Что внутри

15 тем (все лекции 2026): введение · kNN · линейная регрессия · градиентный спуск ·
регуляризация · отступ и функции потерь · логистическая регрессия · SVM ·
метрики классификации · решающие деревья · бэггинг/случайный лес · бустинг ·
кластеризация · снижение размерности · ранжирование.

~127 карточек теории · ~254 теста · ~184 открытых вопроса · ~204 формулы в шпаргалке.

### 4 режима
- **Теория** — карточки: интуиция → формула → вывод → почему именно так → связи → типичные ошибки.
- **Тест** — MCQ с пояснением каждого варианта (позиции ответов перемешаны).
- **Разбор формул** — открытые вопросы в стиле КР («объясни смысл отступа», «распиши шаг SGD»,
  «найди ошибки в псевдокоде») с эталонным ответом и самооценкой (интервальное повторение).
- **Шпаргалка** — все формулы + расшифровка каждого символа.

Формулы рендерятся через встроенный MathJax (tex-svg), поэтому файл полностью автономен.

## Как пересобрать

Контент лежит в `build/topics/*.json` (по одному файлу на тему). Сборка:

```bash
cd build
python3 build_html.py          # склеивает topics/*.json + template.html + MathJax → index.html
```

Доп. скрипты: `shuffle_quiz.py` (балансировка позиций ответов), `merge_enew.py`
(вмерживание переразмеченных пояснений), `methodology.md` (методический гайд авторинга).
